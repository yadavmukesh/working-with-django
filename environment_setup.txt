#For setting up environment for windows machine
Steps:
1. Download python 2.7.x
2. Install python 2.7.x
3. Run command prompt
4. To check python installed version
    python --version
5. Install git
6. To check git version
    git --version
7. To make virtual environment first install virtualenvironment
    pip install virtualenv
8. To make a virtual environment named "env"
    virtualenv --no-site-packages env
9. To activate virtual environment (use this to get into virtual environment)
    env\scripts\activate
10. To close or deactivate virtual environment
    deactivate
11. To install django
    pip install django==1.5
12. cmd->python
14. cmd->import django
15. cmd->django.get_version()
